# Zenn

## zenn-cli

- [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

```sh
$ npx zenn init

  🎉  Done!
  早速コンテンツを作成しましょう

  👇  新しい記事を作成する
  $ zenn new:article

  👇  新しい本を作成する
  $ zenn new:book

  👇  投稿をプレビューする
  $ zenn preview
```

## textlint

特定のファイルに対して textlint を実行する

```sh
npm run lint -- ./articles/[slug].md
```
