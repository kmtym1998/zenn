# はじめに

Raycast とは、Mac OS で使えるランチャーアプリです。類似する機能を持つ製品としては [Alfred](https://www.alfredapp.com/) や Apple 純正の Spotlight などがあります。

https://www.raycast.com/

自分は Raycast を仕事でもプライベートでも使っています。
